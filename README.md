# Inchicore Contracting Inc. Website

This is the concept website for **Inchicore Contracting Inc.**, a Toronto-based home renovation and custom building contractor specializing in East Toronto and Scarborough.

## 🚧 Purpose

- Present a modern, single-page design showcasing services, past projects, contact info, and lead generation tools.
- Allow for private collaboration during the planning and approval stages.
- Designed for deployment via GitHub Pages when ready.

## 🔍 Project Structure

```
inchicore-site/
├── index.html         # Main single-page website
├── assets/            # Future CSS or JS files
├── images/            # Photo gallery and project images
├── scripts/           # Custom JS if needed
├── README.md          # This file
└── .gitignore         # (optional) Git ignore rules
```

## 📦 Features

- Tailwind CSS for fast, responsive design
- Feather icons and Animate.css for visual polish
- Embedded contact form and map
- Email subscribe form
- Smooth scroll and animations
- Optimized for GitHub Pages
